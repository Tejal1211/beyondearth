# Beyond Earth – Parallax Scrolling Website

Beyond Earth is a futuristic one-page website designed to deliver an immersive storytelling experience through parallax scrolling and interactive animations.

## Features
- 🚀 Smooth parallax scrolling
- 🌌 Cinematic space-themed visuals
- ✨ Glassmorphism UI
- 🎨 Responsive modern design
- 🌠 Animated stars and floating elements
- 📱 Mobile-friendly layout
- ⚡ Built with HTML, CSS, and JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript
